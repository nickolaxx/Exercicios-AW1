K02 - Desenvolvimento Web com HTML, CSS e JavaScript

Atualmente, praticamente todos os sistemas corporativos possuem interfaces wed. Para quem deseja atuar no mercado de desenvolvimento de software, é obrigatorio o conhecimento das linguagens: HTML, CSS e Javascript.

Essas linguagens são utilizados para o desenvolvimento da camada de apresentação das aplicações web.

Pré-requisitos

  • Familiaridade com algum sistema operacional(Windows/Linux/Mac 0s x)
  • Lógica de programação
